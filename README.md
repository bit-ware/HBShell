# HBShell

Simple but powerful replacement for ./hbase shell

## Installation

Execute:

    $ wget http://nodeload.github.com/xxjapp/HBShell/zip/master
    $ unzip master
    $ mv HBShell-master HBShell
    $ cd HBShell/

## Shell Commands

    $ ruby run.rb
